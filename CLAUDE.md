# 开发说明

本仓库是 AGENTS 框架的源码。框架定义在 `AGENTS.md`，安装后复制到 `~/.claude/CLAUDE.md`。

开发时请直接修改项目文件（`rules/`、`skills/`、`agents/`、`AGENTS.md`），用户负责同步到 `~/.claude/`。

## 提示词视角检查清单

修改 Rules/Skills/提示词文件时逐项检查：
- 指令使用祈使语气，非描述性语句
- 无歧义：每条指令只有一种合理解读
- 无冗余：不重复其他文件已定义的规则（引用代替复制）
- 可验证：指令执行结果可通过客观标准判断
- 无冗余引用：X 已被自动加载时，禁止写"参见 X 文件"
